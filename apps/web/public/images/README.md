## Creating Placeholder Images

For development, you can create simple placeholders using:
- **Online tools**: Canva, Figma, Adobe Express
- **AI Generation**: Midjourney, DALL-E, Stable Diffusion
- **Stock photos**: Unsplash, Pexels (modified with text/graphics)

## Image Optimization

Before adding to production:
1. Optimize all images using tools like TinyPNG or Squoosh
2. Export as PNG for illustrations with transparency
3. Use WebP format for better compression
4. Ensure proper sizing to avoid loading large files

## Notes

- All images should use warm, friendly colors (orange, yellow, green, blue)
- Focus on Indonesian cultural context (hijab representation, local settings)
- Keep designs simple and non-technical for "emak-emak" target audience
- Use happy, confident expressions in all illustrations
